== Description ==

To keep long life cookies post ITP2.1, cookies must have the attributes Httponly and Secure. These can only be set server-side, so I built this small AJAX plugin to ensure your ads and analytics cookies get upgraded for a longer shelf life. This is only an MVP.

== Installation ==

1. Upload `secure-httponly-tracking` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
